# 프로젝트 기획서

## 1. 프로젝트 개요
### 1. 프로젝트 주제: **중소기업과 구직자의 Matching 플랫폼**

- 직무역량 확보방안
- 유망 중소기업 추천
- 기업 인채추천 및 Direct Job Offering
- 커리어패스 리포트
